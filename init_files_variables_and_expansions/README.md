# Shell, init files, variables and expansions

Este proyecto se enfoca en el uso de variables de entorno y locales, expansiones aritméticas, conversiones de base y configuración del entorno en la shell (Bash).

## Repositorio
* **Repositorio de github:** `holbertonschool-shell`
* **Directorio:** `init_files_variables_and_expansions`

## Tasks List

* **0. <o>:** Crea un alias llamado `ls` que ejecuta `rm -f *`.
* **1. Hello you:** Imprime `hello user`, donde user es el usuario actual de Linux.
* **2. The path to success...:** Añade el directorio `/action` al final de la variable de entorno `PATH`.
* **3. If the path be beautiful...:** Cuenta el número de directorios que hay en la variable `PATH`.
* **4. Global variables:** Lista todas las variables de entorno globales.
* **5. Local variables:** Lista todas las variables (locales, globales) y funciones.
* **6. Local variable:** Crea una variable local llamada `BEST` con el valor `School`.
* **7. Global variable:** Crea una variable global (entorno) llamada `BEST` con el valor `School`.
* **8. Every addition to true knowledge...:** Imprime la suma de 128 más el valor de la variable de entorno `TRUEKNOWLEDGE`.
* **9. Divide and rule:** Imprime el resultado de dividir la variable `POWER` entre `DIVIDE`.
* **10. Love is anterior to life...:** Calcula y muestra el resultado de `BREATH` elevado a la potencia de `LOVE`.
* **11. There are 10 types of people...:** Convierte un número en base 2 (almacenado en `BINARY`) a base 10 (decimal).
* **12. Combination:** Imprime todas las combinaciones posibles de dos letras minúsculas (de la 'a' a la 'z'), ordenadas alfabéticamente, excluyendo "oo".
* **13. Floats:** Imprime un número (almacenado en `NUM`) formateado con exactamente dos decimales.
* **14. Decimal to Hexadecimal:** Convierte un número en base 10 (almacenado en `DECIMAL`) a base 16 (hexadecimal).
